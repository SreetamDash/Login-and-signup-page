# 🔐 Login & Signup Page — Day 2 of #10DaysOfWebDev

A clean and fully functional Login & Signup form built with HTML, CSS, and JavaScript.

---

## 🔗 Links

- **Live Demo:** [https://loginsignup002.netlify.app/]
- **GitHub Repo:** []



## ✨ Features

- Signup form with full validation
- Login form with credential check
- Real-time error messages for each field
- Password match confirmation
- User data stored using `localStorage`
- Smooth fade-in card animation
- Clean gradient UI design
- Responsive layout

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

---

## 📁 Project Structure

```
Signup-page/
├── index.html
├── signup.html
├── style.css
└── script.js
```

---

## 🚀 How to Run

1. Clone the repo or download the files
2. Open `signup.html` to create an account
3. You'll be redirected to `index.html` after signup
4. Login with the same credentials
5. No setup or server needed!

---

## ✅ Validations Covered

| Field | Rule |
|---|---|
| Full Name | Minimum 3 characters |
| Email | Must match valid email format |
| Password | Minimum 6 characters |
| Confirm Password | Must match the password field |

---

## 💡 What I Learned

- Form validation using JavaScript
- Regex pattern for email validation
- Storing and retrieving data with `localStorage`
- Handling multiple forms in a single JS file using `if` checks

---

## ⚠️ Note

Passwords are stored as plain text in `localStorage` for learning purposes. In a real-world app, passwords should always be hashed and stored securely on a backend server.
